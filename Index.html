02 오늘 등록된 일정

아직 등록된 일정이 없습니다

설정

API 키는 이 브라우저에만 저장되며 외부로 전송되지 않습니다.

처음 설정하시나요? (열어서 순서 확인)
  1. Anthropic 키: console.anthropic.com → API Keys → 새 키 생성 후 위에 붙여넣기
  2. Google Cloud Console(console.cloud.google.com)에서 새 프로젝트 생성
  3. 「API 및 서비스 → 라이브러리」에서 Google Calendar API 사용 설정
  4. 「OAuth 동의 화면」에서 외부 선택 → 앱 이름 입력 → 본인 이메일을 테스트 사용자로 추가
  5. 「사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID」
    애플리케이션 유형: 웹 애플리케이션
  6. 승인된 자바스크립트 원본에 이 앱을 여는 주소를 정확히 추가
    (예: GitHub Pages라면 https://아이디.github.io)
  7. 생성된 클라이언트 ID를 위 칸에 붙여넣기
